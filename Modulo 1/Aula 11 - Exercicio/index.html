HTML

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar a estrutura e o conteúdo de páginas da web. Ele usa elementos e tags para identificar diferentes tipos de conteúdo, como títulos, parágrafos, imagens, links, formulários e muito mais. A estrutura básica de um documento HTML consiste em uma tag html que contém cabeçalho e corpo. Dentro do corpo, os elementos são organizados hierarquicamente, permitindo criar a estrutura e o layout da página

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o design de um documento HTML. Ele permite controlar a aparência dos elementos HTML, como cores, fontes, tamanhos, espaçamento, posicionamento e efeitos visuais. Com o CSS, você pode criar layouts responsivos, estilizar botões, aplicar animações e muito mais. Os estilos CSS podem ser definidos internamente em um documento HTML, incorporados usando a tag style ou ainda externamente em um arquivo separado e vinculado ao documento HTML através da tag

JAVASCRIPT

JavaScript é uma linguagem de programação de alto nível e multiplataforma que permite adicionar interatividade e funcionalidade a páginas da web. Com o JavaScript, você pode manipular elementos HTML, responder a eventos, criar animações, validar formulários, fazer requisições a servidores, entre muitas outras coisas. Ele é executado no navegador do usuário, permitindo a criação de experiências interativas e dinâmicas. Além de ser usado no desenvolvimento front-end, o JavaScript também pode ser executado no lado do servidor Node.js e em aplicativos móveis usando frameworks como React Native e Ionic
